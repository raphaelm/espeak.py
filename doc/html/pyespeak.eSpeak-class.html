source code 

Initializes an eSpeak process

Parameters:
  • voice (string) - Voice to use (e.g. 'en' or 'de')
  • amplitude (int) - Amplitude, 0 to 200, default is 100
  • gap (int) - Word gap. Pause between words, units of 10mS at the default speed
  • capitals (int) - Indicate capital letters with: 1=sound, 2=the word "capitals", higher values indicate a pitch increase.
  • linelength (int) - Line length. If not zero (which is the default), consider lines less than this length as end-of-clause
  • pitch (int) - Pitch adjustment, 0 to 99, default is 50
  • speed (int) - Speed in words per minute, 80 to 450, default is 175
  • markup (bool) - Interpret SSML markup, and ignore other < > tags
  • nofinalpause (bool) - No final sentence pause at the end of the text
  • path (string) - Specifies the directory containing the espeak-data directory
  • punct (string) - Speak the names of punctuation characters during speaking. If this is True, all punctuation is spoken.

reopen(self)

source code 

Waits until everything in the queue is spoken and then closes the process and opens an new one.

Unfortunately there doesn't seem to be any other way to wait for the end of the audio output without terminating the process.
